# Project Page Assets

The project page combines paper figures with optimized showcase images.

Paper figures:

```text
overview.png
main_qual_corrected.png
main_qual_corrected.pdf
```

Showcase assets under `showcase/` are compressed JPEG images built from selected
supplemental `new_pic/` content-reference-result examples. The public page uses
a curated lead grid plus an eight-case carousel. Images are displayed with
complete content/reference evidence beside larger results, avoiding overlay
cards and awkward cropping. The README uses `readme_spotlight.jpg` as a compact
static preview of the lead grid.

PDF figures are kept only as source/download assets. Browser-visible figures use
PNG or JPEG images for stable rendering.
